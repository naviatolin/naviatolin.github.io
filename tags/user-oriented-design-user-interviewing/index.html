Designing Tools For Safe Consumption Site Advocates

I worked with safe consumption site advocates in the Boston area to design a tool that could potentially revolutionize the current state of safe consumption sites in the U.S. today. This project was done in a team of four and over a semester through a user oriented collaborative design class. The severity of the opioid epidemic today was shocking to see and we all really wanted to do something about it.
