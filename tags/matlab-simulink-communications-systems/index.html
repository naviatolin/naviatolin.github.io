Dec. 15, 2020

Complete Communications System Modeling Using BPSK

I leveraged introductory-level skills in analog-digital communication (ADC) to design an end-to-end communication system in Simulink. The Simulink model of the system was used to then select appropriate physical components and understand a potential hardware implementation. The ADC system model was designed to use Binary Phase Shift Keying (BPSK) as the modulation scheme and optimized to fit within a handful of real-world constraints including power limitations, transmission rate, error rate, and availability of components for the hardware implementation.
